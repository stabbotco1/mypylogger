# mypylogger Documentation

This directory contains comprehensive documentation for the mypylogger project.

## Setup Guides

- **[GitHub Token Setup](GITHUB_TOKEN_SETUP.md)** - Complete guide for setting up GitHub Personal Access Token for pipeline monitoring

## Features

### Core Logging Features
- Structured JSON logging with consistent format
- Environment-driven configuration
- Real-time development support with immediate log flushing
- Thread-safe singleton pattern
- Graceful error handling

### Advanced Development Tools
- **GitHub Actions Pipeline Monitoring** - Real-time monitoring of CI/CD pipeline status
- **Local Development Integration** - Seamless integration with local development workflows
- **Quality Gate Integration** - Pipeline status integration with local test suites
