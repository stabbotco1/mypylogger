## GitHub Actions Pipeline Fix

**Issue**: Upload Coverage job failing with "Input required and not supplied: aws-region"

**Root Cause**: The `upload-coverage` job was missing the `aws-region` parameter in the AWS credentials configuration step, while other jobs (deploy, notify-status) had it correctly configured.

**Fix Applied**: Added missing `aws-region: ${{ vars.AWS_REGION }}` parameter to the `Configure AWS credentials via OIDC` step in the `upload-coverage` job.

**Status**: ✅ Fixed - Pipeline should now pass the AWS credentials configuration step
