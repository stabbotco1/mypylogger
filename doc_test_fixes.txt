Basic Usage\n\n```bash\n# Monitor current commit\'s pipeline status\npython scripts/github_pipeline_monitor.py --status-only\n\n# Monitor and wait for pipeline completion\npython scripts/github_pipeline_monitor.py\n\n# Monitor after pushing to pre-release branch\npython scripts/github_pipeline_monitor.py --after-push --branch pre-release\n```\n\n### 3. Verify your GitHub token is set: `echo $GITHUB_TOKEN`\n2. Check token permissions include "Actions: Read"\n3. Regenerate token if necessary\n\n**Problem**: `403 Forbidden` error\n```\n❌ Error: GitHub API access denied - insufficient permissions\n```\n\n**Solution**:\n1. Ensure token has "Actions: Read" permission\n2. Verify you have access to the repository\n3. Check if repository is private and token has appropriate scope\n\n#### Repository Detection Issues\n\n**Problem**: Repository not found or not accessible\n```\n❌ Error: Repository owner/repo not found or not accessible\n```\n\n**Solution**:\n1. Verify git remote is configured: `git remote -v`\n2. Check repository name and owner are correct\n3. Ensure you have read access to the repository\n\n#### Rate Limiting\n\n**Problem**: Rate limit exceeded\n```\n⚠️ GitHub API rate limit warning: 5 requests remaining\n```\n\n**Solution**:\n1. Wait for rate limit to reset (shown in warning)\n2. Reduce polling frequency with `--poll-interval`\n3. Use caching to reduce API calls\n\n#### Network Issues\n\n**Problem**: Network connectivity errors\n```\n❌ Error: Network connectivity error: [Errno 11001] getaddrinfo failed\n```\n\n**Solution**:\n1. Check internet connection\n2. Verify GitHub API is accessible: `curl https://api.github.com`\n3. Check firewall/proxy settings\n\n### Debug Mode\n\nEnable verbose output for detailed troubleshooting:\n\n```bash\n# Verbose output with detailed information\npython scripts/github_pipeline_monitor.py token is set\necho $GITHUB_TOKEN\n\n# Check shell profile\ngrep GITHUB_TOKEN ~/.bashrc ~/.zshrc ~/.bash_profile 2>/dev/null\n```\n\n**Solutions:**\n\n1. **Set the token in your shell profile:**\n   ```bash\n   echo \'export GITHUB_TOKEN="your_token_here"\' >> ~/.zshrc\n   source ~/.zshrc\n   ```\n\n2. **Set temporarily for testing:**\n   ```bash\n   export GITHUB_TOKEN="your_token_here"\n   python scripts/github_pipeline_monitor.py --status-only\n   ```\n\n3. **Use configuration file:**\n   ```yaml\n   # .github-monitor.yml\n   github:\n     token: your_token_here\n   ```\n\n### Problem: "GitHub API authentication failed - invalid or missing token"\n\n**Error Message:**\n```\n❌ Error: GitHub API authentication failed - invalid or missing token\n```\n\n**Diagnosis:**\n```bash\n# Test token manually\ncurl -H "Authorization: token $GITHUB_TOKEN" https://api.github.com/user\n```\n\n**Solutions:**\n\n1. **Verify token format:**\n   - Token should start with `ghp_`\n   - Should be 40+ characters long\n\n2. **Check token hasn\'t expired:**\n   - Go to GitHub Settings → Developer settings → Personal access tokens\n   - Check expiration date\n\n3. **Regenerate token:**\n   - Create new token with same permissions\n   - Update environment variable\n\n### Problem: "GitHub API access denied - insufficient permissions"\n\n**Error Message:**\n```\n❌ Error: GitHub API access denied - insufficient permissions\n```\n\n**Diagnosis:**\n```bash\n# Check token permissions\ncurl -H "Authorization: token $GITHUB_TOKEN" https://api.github.com/repos/OWNER/REPO/actions/runs\n```\n\n**Solutions:**\n\n1. **Verify token permissions:**\n   - Go to GitHub Settings → Developer settings → Personal access tokens\n   - Ensure "Actions: Read" is selected\n\n2. **Check repository access:**\n   - Verify you have read access to the repository\n   - For private repos, ensure token has appropriate scope\n\n3. **Organization settings:**\n   - Some organizations restrict personal access tokens\n   - Contact organization admin if needed\n\n## Configuration Problems\n\n### Problem: "Repository not found or not
